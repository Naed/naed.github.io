Markdown基础语法

目录

[TOC]表示创建目录列表，代码格式: [TOC]。在Vscode中使用Markdown TOC自动生成的插件。

  • 目录
  • 标题
  • 换行
  • 分割线
  • 引用
  • 文字样式
  • 代码高亮
  • 列表
  • 待办事项
  • 链接以及图片
  • 表格
  • 流程图
  • 甘特图
  • 来源参考

标题

Markdown支持6种级别的标题，对应html标签 h1 ~ h6

代码格式：# 标题内容

换行

换行标签 </br>

Test1

Test2

分割线

三个或者三个以上的 - 或者 *

引用

> 引用话语

文字样式

在Markdown中有：斜体、加粗、高亮、划线。

斜体 加粗 ==高亮== 划线



如需修改文字大小/颜色/字体，就要用font标签

color代表字体颜色（要用16进制颜色值），size代表文字大小，face代表字体

宋体大小为12的字

代码高亮

key有对应的编程语言

常用编程语言:(基本是小写形式)

  • CSS - css
  • HTML - html
  • javaScript - javascript
  • JSON - json
  • Less - less
  • SCSS - scss
  • YAML - yml
  • text - text/plain

列表

在Markdown中可以绘制列表

  • 列表1
    • 列表1.1
    • 列表1.2
      • 列表1.21
      • 列表1.22

待办事项

在Markdown中输入你最近的代办事项

  • 未完成事项
    • 未完成事项1
    • 未完成事项2
  • 已完成事项
    • 已完成1
