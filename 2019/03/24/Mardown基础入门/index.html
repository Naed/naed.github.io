文字样式

在Markdown中有：斜体、加粗、高亮、划线。

斜体 加粗 ==高亮== 划线



如需修改文字大小/颜色/字体，就要用font标签

color代表字体颜色（要用16进制颜色值），size代表文字大小，face代表字体

代码高亮

key有对应的编程语言

常用编程语言:(基本是小写形式)

  • CSS - css
  • HTML - html
  • javaScript - javascript
  • JSON - json
  • Less - less
  • SCSS - scss
  • YAML - yml
  • text - text/plain

列表

  • 列表1
    • 列表1.1
    • 列表1.2
      • 列表1.21
      • 列表1.22

待办事项

在Markdown中输入你最近的代办事项

  • 未完成事项
    • 未完成事项1
    • 未完成事项2
  • 已完成事项
